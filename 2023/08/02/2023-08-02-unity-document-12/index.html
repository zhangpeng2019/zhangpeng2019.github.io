Unity界面-项目窗口 🔗

项目窗口显示与项目相关的所有文件，是您在应用程序中导航和查找资源及其他项目文件的主要方式。默认情况下，在开始新项目时，该窗口是打开的。但是，如果您找不到它或关闭了它，您可以通过窗口 > 常规 > 项目或按Ctrl+5（macOS：Cmd+5）打开它。

您可以通过点击窗口顶部并拖动来移动项目窗口。您可以将其停靠在编辑器中的特定位置，或将其拖出编辑器窗口以使用它作为自由浮动窗口。您还可以更改窗口本身的布局。要做到这一点，选择窗口右上角的更多菜单（⋮），然后从一列布局或两列布局中进行选择。两列布局具有额外的窗格，显示每个文件的可视化预览。

浏览器的左侧面板显示项目的文件夹结构，以层级列表的形式呈现。当您从列表中选择一个文件夹时，Unity会在右侧窗格中显示其内容。您可以点击小三角形来展开或折叠文件夹，显示其包含的任何嵌套文件夹。按住Alt键并单击可以递归地展开或折叠任何嵌套文件夹。

单个资源显示在右侧面板中，以图标的形式表示其类型（例如，脚本、材质、子文件夹）。要调整图标的大小，使用面板底部的滑块；如果将滑块移至极左侧，它们将被替换为层级列表视图。滑块左侧的空间显示当前选择的项，包括项目正在进行搜索时的完整路径。

在项目结构列表的上方是一个收藏夹部分，您可以将常用项保存在那里以便快速访问。您可以将项目结构列表中的项拖动到收藏夹中，并且还可以将搜索查询保存在那里。

项目窗口工具栏 🔗

在窗口的顶部边缘是浏览器的工具栏。

Property Description
创建菜单 显示一个可以添加到当前选定文件夹的资源和子文件夹的列表。
搜索栏 使用搜索栏在项目中搜索文件。您可以选择在整个项目(全部)、项目的顶层文件夹（逐个列出）、当前选定的文件夹内或资产商店中搜索文件。
在搜索中打开 打开Unity搜索工具以进一步细化搜索。
按类型搜索 选择此属性将限定搜索到特定类型，例如网格、预设、场景等。
按标签搜索 选择此属性选择要在其中搜索的标签。
保存搜索 将您的搜索保存在左侧面板的收藏夹中。
隐藏的包数量 选择此属性以切换在项目窗口中显示的包的可见性。

搜索过滤器 🔗

搜索过滤器通过在搜索文本中添加一个额外的项来起作用。以t:开头的项按指定的资源类型进行过滤，而以l:开头的项按标签进行过滤。如果您知道自己要查找的内容，可以直接在搜索框中输入这些项，而不必使用菜单。您可以同时搜索多个类型或标签。添加多个类型会扩展搜索以包括所有指定的类型（即，类型之间进行OR操作）。添加多个标签也会扩展搜索以包括所有指定的标签（即，标签之间进行OR操作）。如果您搜索类型和标签，搜索结果将返回与类型和标签都匹配的资源（即，类型和标签之间进行AND操作）。

快捷键 🔗

在浏览器视图具有焦点时，可以使用以下键盘快捷键。请注意，其中一些快捷键仅在视图使用双列布局时才有效（您可以使用右上角的面板菜单在单列布局和双列布局之间切换）。

快捷键 功能
F 定位选择的项（即，在包含选定资源的文件夹中显示选定资源）
Tab 在第一列和第二列之间切换焦点（双列布局）
Ctrl/Cmd + F 聚焦搜索字段
Ctrl/Cmd + A 选择列表中所有可见项
Ctrl/Cmd + D 复制选定的资源
Delete 带有对话框的删除（Windows）
Delete + Shift 不带对话框的删除（Windows）
Delete + Cmd 不带对话框的删除（OSX）
Enter 开始重命名选定项（OSX）
Cmd + down arrow 打开选定的资源（OSX）
Cmd + up arrow 跳转到父文件夹（OSX，双列布局）
F2 开始重命名选定项（Windows）
Enter 打开选定的资源（Windows）
Backspace 跳转到父文件夹（Windows，双列布局）
Right arrow 展开选定的项（树视图和搜索结果）。如果该项已展开，则会选择其第一个子项。
Left arrow 折叠选定的项（树视图和搜索结果）。如果该项已折叠，则会选择其父项。
Alt + right arrow 在预览中显示资源时展开项
Alt + left arrow 在预览中显示资源时折叠项
