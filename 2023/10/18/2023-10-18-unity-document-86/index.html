项目设置 🔗

使用 “项目设置” 窗口来定义项目的设置。要打开 “项目设置” 窗口，请转到 Edit > Project Settings。

A 搜索框允许您在左侧筛选设置类别列表，并在右侧的详细面板中突出显示关键字。

B 所有设置按类别组织，类别显示在左侧的类别列表中。当您从列表中选择一个类别时，详细面板会显示关联的设置。

C 详细面板显示所选类别的所有可用设置。

下面列出了 “项目设置” 窗口中默认提供的设置类别。

  • Adaptive Performance
  • Audio
  • Editor
  • Graphics
  • Input Manager
  • Package Manager
  • Physics
  • Physics 2D
  • Player
  • Preset Manager
  • Quality
  • Scene Template
  • Script Execution Order
  • Services
  • Tags and Layers
  • TextMesh Pro
  • Time
  • Timeline
  • UI Toolkit
  • Version Control
  • XR Plugin Management

根据您安装的包，项目设置窗口还可以显示其他类别和设置。有关特定于包的设置的更多信息，请参阅相应的包文档。

查找设置 🔗

当您需要找到一个设置时，您可以在 “项目设置” 窗口顶部的搜索框中输入一个关键字（可以是部分或全部关键字）。

通过在搜索框中输入关键字来限定范围

当您在搜索框中输入关键字时，类别列表会动态变化。在上面的示例中，当用户输入单词 “mode” 时，类别列表会缩小为六个匹配项。这意味着 Unity 在仅六个范围类别的项目设置中找到了该关键字。同时，详细面板会突出显示任何匹配的关键字。

提示：您可以使用相同的步骤来查找和过滤首选项。
