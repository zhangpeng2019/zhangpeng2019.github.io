Audio 🔗

Audio设置（主菜单：Edit > Project Settings，然后选择Audio类别）允许您微调场景中所有正在播放的声音的最大音量。

Property Function
Global Volume 设置播放期间所有声音的音量。
Volume Rolloff Scale 设置基于对数衰减的源的全局衰减系数。值越大，音量衰减越快。相反，值越小，衰减越慢。提示：1 的值模拟 “真实世界”。
Doppler Factor 设置多普勒效应的听觉效果。使用 0 禁用。使用 1 使其对快速移动的对象可听。提示：将Doppler Factor设置为 1 后，您可以微调Speed of Sound和Doppler Factor直到满意为止。
Default Speaker Mode 设置项目的默认扬声器模式。默认值为 2，对应立体声扬声器。有关模式的完整列表，请参阅 AudioSpeakerMode API 参考。注意：您还可以通过脚本在运行时更改扬声器模式。有关详细信息，请参阅 Audio Settings。
System Sample Rate 设置输出采样率。如果设置为 0，Unity 将使用系统的采样率。注意：这只是一个参考，因为某些平台允许您更改采样率，例如 iOS 或 Android。
DSP Buffer Size 设置 DSP 缓冲区的大小，以优化延迟或性能。
Default 默认缓冲区大小。
Best Latency 在性能和延迟之间权衡。
Good Latency 在延迟和性能之间取得平衡。
Best Performance 在性能和延迟之间权衡。
Max Virtual Voices 设置音频系统管理的虚拟声音数的最大数量。该值应始终大于游戏播放的声音数量。如果不是，Unity 会在控制台显示警告。
Max Real Voices 设置同时播放的真实声音数的最大数量。在每一帧中选择最响亮的声音。
Spatializer Plugin 选择要使用的本地音频插件，以对 3D 声源执行空间化滤波。
Ambisonic Decoder Plugin 选择要使用的本地音频插件，以执行源的全景声到双声道滤波。
Disable Unity Audio 启用后，禁用独立构建中的音频系统。在编辑器中，音频系统仍然开启，并支持预览音频剪辑，但 Unity 不处理对 AudioSource.Play 和 AudioSource.playOnAwake 的调用，以模拟独立构建的行为。
Virtualize Effect 启用后，动态关闭在被剔除的 AudioSource 上的效果和空间化器，以节省 CPU。
