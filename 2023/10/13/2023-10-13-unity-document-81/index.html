2D 和 3D 模式设置 🔗

在创建新项目时，您可以指定 Unity 编辑器是以 2D 模式还是 3D 模式启动。然而，您也可以随时在 2D 模式和 3D 模式之间切换编辑器。您可以在此处详细了解 2D 和 3D 项目之间的区别。本页面提供了有关如何切换模式以及切换模式时编辑器中发生的具体变化的信息。

在 3D 和 2D 模式之间切换 🔗

要在 2D 模式和 3D 模式之间切换，请按照以下步骤进行操作：

  1. 打开编辑器设置（顶部菜单：Edit > Project Settings，然后选择Editor类别）。
  2. 然后将Default Behavior Mode设置为2D或3D。

使用编辑器设置中的默认行为模式设置将项目设置为 2D 或 3D。

2D 模式与 3D 模式设置 🔗

2D 模式和 3D 模式确定了 Unity 编辑器的一些设置。以下是这些设置的详细列表。

在 2D 项目模式下： 🔗

  • 导入的任何图像都被认为是 2D 图像（Sprites），并设置为Sprite模式。
  • 场景视图设置为 2D。
  • 默认游戏对象没有实时定向光。
  • 相机的默认位置为 0,0,-10。（在 3D 模式下为 0,1,-10。）
  • 相机设置为正交投影。（在 3D 模式下为透视投影。）
  • 在光照窗口中：
    • 对于新的场景，天空盒已禁用。
    • 环境源设置为颜色（颜色设置为深灰色：RGB：54，58，66）。
    • 实时全局光照（Enlighten）已关闭。
    • 烘焙全局光照已关闭。
    • 自动构建已关闭。

在 3D 项目模式下： 🔗

  • 导入的任何图像都不会被认为是 2D 图像（Sprites）。
  • 场景视图设置为 3D。
  • 默认游戏对象具有实时定向光。
  • 相机的默认位置为 0,1,-10。（在 2D 模式下为 0,0,-10。）
  • 相机设置为透视投影。（在 2D 模式下为正交投影。）
  • 在光照窗口中：
    • 天空盒是内置的默认天空盒材质。
    • 环境源设置为天空盒。
    • 实时全局光照（Enlighten）已开启。
    • 烘焙全局光照已开启。
    • 自动构建已开启。
