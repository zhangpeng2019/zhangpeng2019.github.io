翻译Unity中文版的初衷是因为官方提供的中文版存在缺陷，而且翻译的不全。现在基于Unity2023.2版本对官方文档进行翻译。

UI Toolkit project settings 🔗

要查看 UI Toolkit 项目设置，请转到Edit > Project Settings > UI Toolkit。

UI Builder 🔗

在 UI Builder 中创建新的UI文档（UXML）时，默认设置了某些功能：

  • UI 文档被设置为适用于运行时 UI。
  • UI Builder 库仅显示适用于运行时 UI 的 UI 控件。
  • 您可以使用鼠标滚轮或轨迹板轻松缩放 UI Builder 中视口窗口的画布。

为了确保新创建的文档默认与编辑器 UI 兼容，并且 UI Builder 库显示用于编辑器 UI 的其他控件，请选择默认启用编辑器扩展编写。有关更多信息，请参阅为 UI 文档（UXML）启用编辑器扩展编写。

为了禁用使用鼠标滚轮或轨迹板缩放画布，请选择禁用鼠标滚轮 / 轨迹板缩放视口。

高级 🔗

UI Toolkit 事件调试器是一个实验性功能，可以帮助您了解如何处理各种 UI Toolkit 事件，例如输入、布局和绘制。UI Toolkit 事件调试器窗口默认处于禁用状态。要启用它，请选择启用事件调试器。一旦启用，可以通过选择Windows > UI Toolkit > Event Debugger来访问它。

UI Toolkit 布局调试器是一种实验性功能，可以帮助您了解哪些样式和设置会影响屏幕上的元素。UI Toolkit 布局调试器窗口默认处于禁用状态。要启用它，请选择启用布局调试器。一旦启用，可以通过选择 Windows > UI Toolkit > Layout Debugger来访问它。
